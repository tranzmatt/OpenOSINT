NAME

openosint-sponsors — media kit for vendors reaching OSINT practitioners, security researchers, and developers adopting intelligence APIs

DESCRIPTION

OpenOSINT is a modular OSINT framework with four interfaces: an interactive AI REPL, a direct CLI, an MCP server for Claude Code and Claude Desktop, and a browser-based Web UI. Behind the scenes, 16 intelligence tools wrap external APIs and binaries. When a user configures a new integration, the documentation routes them directly to the provider's sign-up or pricing page.

A Featured Integration sponsor is named as the recommended, default provider for one tool category — and no competitor gets that placement while you hold it. One vendor per category.

REFERRAL FUNNEL

The user journey from install to paying customer is four steps:

  1. Install: pip install openosint or clone from GitHub.
  2. Configure: user reads the Configuration table or tool documentation. Your API is listed as the recommended provider with a direct sign-up link.
  3. Click-through: user navigates to your pricing or sign-up page.
  4. Activate: user sets an API key and the integration runs live.

Users arrive motivated: they have already installed an OSINT framework and are actively looking for a provider. This is a high-intent audience, not a general-readership blog.
